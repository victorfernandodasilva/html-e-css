Principais formatações

Negrito / Destaque

Nesta frase temos um exemplo de negrito usando a tag <B> (Não semântica, só tem forma não significado)

E nesta frase temos um termo em negrito usando a tag <Strong> (Com semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não semântica, só tem forma não significado)

E nesta frase temos um termo em itálico usando a tag <em> (Com semântica)

Texto marcado / Texto grifado

Também podemos grifar um texto usando a tag <mark>

Texto deletado

Podemos marcar um texto deletado mostrando que ele pode ser lido mas não considerado usando a tag <del>

Texto inserido

Podemos marcar um texto inserido com a tag <ins>, mostrando que ela foi inserida depois

Texto sobrescrito

Para mostrar que um texto foi sobscrevido usamos a tag <sup>

Texto subescrito

Para mostrar que um texto é subescrito usamos a tag <sub>
